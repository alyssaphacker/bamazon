# bamazon

A unique way to purchase items

## Getting Started

Grab all of this code. Install node, the two dependencies, and MySQL

### Setting up Node

Install the project dependencies

```
npm init
npm install inquirer
npm install mysql
```

Then, you should be able to run the app in node

```
node *.js
```
